Website Brief Intro

This website is built up using some of the most popular tools, technologies and frameworks. SublimeText and Emmet are used for maximized typing efficiency. Nodejs and Grunt are used for tasks automation. Instead of using Bootstrap or Foundation for Responsive Web Development(RWD), I used Sass and Susy to get better customization and perfomance. Certain breakpoints are: 625px, 825px, 1075px and 1375px.
